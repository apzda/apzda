# 通用基础模块

| # | 名称        | 服务ID | 说明                                                        |
|:--|:----------|:-----|:----------------------------------------------------------|
| 1 | config    | 900  | 应用设置(首选项)服务                                               |
| 2 | sms       | 901  | 短信服务, 通道支持: 阿里大鱼、创蓝                                       |
| 3 | uc        | 902  | 用户中心，提供用户管理相关功能                                           |
| 4 | wallet    | 903  | 用户钱包                                                      |                                                      
| 5 | audit     | 904  | 审计服务                                                      |
| 6 | captcha   | 905  | 验证码服务                                                     |
| 7 | oss       | 906  | 对象存储服务, 后端已支持: 阿里云OSS、腾讯云COS、MinIO和本地存储。内置插件: 添加水印和Resize |
| 8 | messenger | 908  | 信使服务                                                      |
